# Baby Theremin Animals — README (v2.0.0)

どうぶつの童謡と一緒に遊べる、キッズ向けビジュアル・テルミンです。  
起動するたびに「いぬ・くま・ぞう」のどれかがランダムで選ばれ、曲に合わせてどうぶつのシルエットが画面いっぱいにはじけます。

## 使い方
1. **「はじめる」をタップ**：ランダムなどうぶつの曲が流れ、図形がはじけます。
2. **画面を短くタップ**：効果音 + どうぶつシルエットのバースト。
3. **長押し + 上下ドラッグ**：テルミン（指を離すと停止）。
4. **2本指長押し（約1.1秒）**：操作パネルを表示し、好きな動物（曲）に変更可能。

## 収録ファイル構成
- `index.html`: アプリ本体
- `manifest.json`: PWA設定ファイル
- `sw.js`: オフライン対応用 Service Worker
- `icon-192.png` / `icon-512.png`: アイコン画像
- `Chime.mp3`: タップ時の効果音
