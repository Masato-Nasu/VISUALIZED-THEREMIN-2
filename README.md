# Baby Theremin – Songs & Shapes (v3.2.0)

あなたの v3.1（どうぶつ版）`index.html` をベースに、**「曲が分かる」ことを最優先**に整理したミニマル版です。

## 仕様（削ぎ落とし）
- 起動時：**6曲からランダムで1曲**
- 曲切替：**左下のアイコン**（☆ 🐑 🐻 🦋 🐝 🐸）を押すだけ
- 設定パネル／スライダ／フリップ：**すべて撤去**
- 操作：
  - 短タップ：はじける（効果音あり）
  - 長押し＋上下：テルミン

## 収録
- `index.html`
- `manifest.json`
- `sw.js`
- `icon-192.png` / `icon-512.png`
- `Chime.mp3`

## 公開（GitHub Pages）
1. このフォルダをそのままリポジトリに置く
2. Settings → Pages で branch を選択
3. スマホで開く（必要なら「ホームに追加」）
