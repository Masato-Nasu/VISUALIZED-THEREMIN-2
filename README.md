# Baby Theremin – Songs & Shapes (v3.2.1)

v3.1（あなたの元アプリ）の雰囲気を残しつつ、**混乱要素を削ぎ落として**「曲と形が一瞬で分かる」ことを最優先にした版です。

## 仕様
- 起動時：**6曲からランダムで1曲**
- 曲切替：左下のアイコン（☆ 🐑 🐻 🦋 🐝 🐸）
- 操作：
  - 短タップ：はじける（Chime）
  - 長押し＋上下：テルミン
- 設定パネル／フリップ／スライダ：撤去

## トラブルシュート
- もし「音は出るが画が出ない」場合：v3.2.1 では **中央に絵文字が必ず出る**フォールバックを入れています。
  - それすら出ない場合は、ブラウザのキャッシュを消して再読み込みしてください。
