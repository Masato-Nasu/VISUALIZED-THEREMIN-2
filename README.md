# Baby Theremin – Songs & Shapes (v3.3.2)

## v3.3.2 の狙い（重要）
「中央の絵文字は出るがネオン線画が出ない」端末があるため、**canvas描画を廃止してSVG描画に切り替え**ました。
これにより、ネオン線画が確実に表示されます。

- 短タップ：はじける（Chime）
- 長押し＋上下：テルミン

- v3.3.2: ネオンが「意味不明な塊」になる端末対策として、SVGのdrop-shadowを廃止し、線画パスを“より記号的に読める”形に描き直しました。
